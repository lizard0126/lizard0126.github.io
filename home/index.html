从零开始搭建的个人博客

发表于 2023-12-04 更新于 2023-12-06 分类于 blog 本文字数： 674 阅读时长 ≈ 1 分钟

起因是我有在做本地部署的.js机器人，做了一些社交软件平台的适配。苦于本地部署没法常驻，便购入了云服务器（便宜货。。。）

然而服务器机能在Windows server的系统镜像下已经不够部署运行机器人加QQ，再加上本地的数据库也不太方便转移到云端，遂放弃机器人，转而搭建个人博客/个人论坛

环境的准备

首先注册好github并熟悉基本的操作，很好上手，无非就是新建仓库，找到仓库地址之类，经常使用开源资源的朋友本当上手

其次是在云服安装git，用来绑定github的仓库到本地，创建文件并上传至仓库，其实也可以直接在github编译，但我不会（雾）

然后就是node.js和hexo的云服部署。前者是服务器端的JavaScript运行环境，后者是博客搭建框架，可以很方便地搭建静态网页。但我对markdown这样的标记语言不是很熟悉，还需要学习

关于域名的选购

推荐后缀.top，很便宜，一年只要十块。域名不是必须，不影响博客的云端部署，可以通过ipv4地址或者github地址访问

由于我已经购入了私人域名，需要对域名进行解析，绑定博客的github仓库地址（ipv4地址可以用cmd来ping仓库地址得到，与仓库地址本身共两条解析记录），然后就是漫长而痛苦的修改框架-保存资源-编译-出bug-读日志-改代码-编译-出新bug的无限循环。

碎碎念

对于完全不懂js的我来说，会用命令行已经谢天谢地了。后面可能会更新我对框架的优化，大概率是抄别人的作业，也可能直接发在博客里，更可能直接咕咕咕，这比玩机器人累多了（幸好我给机器人加了gpt，代码问题可以解决）

宿命回响

发表于 2021-12-25 更新于 2023-12-06 分类于 漫研 本文字数： 942 阅读时长 ≈ 1 分钟

2021年12月22日，一场关于音乐与命运的闹剧落下帷幕。南梦宫的游戏企划宣传片«宿命回响takt：op»完结。

回顾整部剧，可以说是一盘好棋下得稀烂，剧情高开低走，前期拖慢进度，中期强补设定，后期剧情崩坏，短短12集挖很多坑却不填，强加意义不明的感情线，最后还是没能听到朝雏磔人的音乐，哼了两句命运就变花瓣了，临死之前哭着献吻，留下变身器让安娜姐在游戏里当命运二代目，磔人则被送去冷藏，延续游戏剧情。

但命运这个角色的塑造无疑是成功的。谙世从不事的怪力三无少女，疯狂摄入甜食，疯狂与指挥家斗嘴，到学着帮助他人，给指挥家“借”钢琴，救下高处的猫，直至最后舍命掩护重伤倒地的指挥家，安慰失去希望的安娜姐，命运的变化直指人心的。奇怪的是这样一个令人印象深刻的角色，南梦宫却选择舍弃，并让安娜姐成为命运的替代，是否可以理解为一种自信呢？

而刀子嘴豆腐心的亚撒西命运酱永远是在默默付出默默承受。无论是第7集暗借钢琴，第8集瞒着磔人对抗地狱，还是选择在磔人失去意识的时候亲吻、告白、离去。这个傲娇的钢琴笨蛋，一定要在游戏剧情里好好把写给命运的情歌演奏出来啊！

动画的细节方面着实可圈可点。而对应op，剧情最后命运将信物交给磔人后消失，这里命运代表奏者身份的眼影消失是否代表着终于成为人类了呢？

神似珂赛特与磔人的路人，是不是某个世界线里没有发生第二集惨剧的what if呢

第6集婆婆教给命运赶走别人的话在第8集有好好用出来，而磔人在第2集的问话“你也懂音乐？”在第12集也有用上。命运与磔人的攻守拌嘴战百看不厌，但即使是游戏剧情里也看不到了。真是寂寞啊。

很可惜，我认为剧情的问题，出在节奏跟反派身上。许多毫无意义的桥段完全可以简化甚至忽略。赌场内的战斗和女武神角色的引入确实可有可无且拖慢节奏。有这些时间为什么不解释一下珂赛特变身石头？为什么不解释一下命运的契约为何如此特别？为什么不解释一下地狱天国救下了年轻的磔人？为什么不解释一下安娜姐的变身？最离谱的是，两个反派的动机是那么的中二且神经病。

12个星期以来，每周的期待，戛然而止时的回味，无不拨动着心弦。除非只看了12集的结尾，否则你大抵都会喜欢上这部剧。只可惜，他成为不了神作，我的感情只有恨铁不成钢。动漫告一段落，就好好期待游戏的制作吧，有缘再会。

2.5次元的诱惑

发表于 2021-11-18 更新于 2023-12-06 分类于 漫研 本文字数： 1.2k 阅读时长 ≈ 1 分钟

我对现实女子完全没有兴趣！

这是来自喜爱着2次元角色・莉莉艾露的漫画研究部部长・奥村的想法。

然而，直到3次元女子・天乃莉莉纱出现后…

为帮助莉莉纱完成梦想而允许对方入部，但没想到的是，

眼前换上cos服后的莉莉纱简直就是——真物（莉莉艾露！？）

这个诱惑不是来自2次元！2.5次元的角色扮演喜剧开幕！

显然，一部校园漫画该有的tag它一个不少：青梅vs天降、废部危机、指导员老师是业界前辈等等。就是这样一部典中典的媚宅漫，却狠狠地吸引了我。虽然已经不是会被南半球冲昏头脑的可爱年纪，但不得不说桥本悠画的欧派是真的吸引人（牛乳乃澪老师并感）

咳咳，扯远了。

一部优秀的漫画，吸引人的地方不仅仅是线条、画功，它的剧情、分镜同样会是加分项。如果单纯靠欧派星人人柱力，是基本没有可能拿到20年的下一部漫画大赏第四位的优异成绩。而这本«2.5次元的诱惑»，可以说是我这两年追更的漫画中最喜欢的漫画之一了。

先说人设。本作的男主奥村，纯纯的莉莉艾露宅。虽然是作者虚构，但从漫画中可以得知莉莉艾露是一个完整的，有漫画动画游戏作品的ip，并且活跃了相当长的时间。相比于不敢触摸宝○梦的莉莉艾，莉莉艾露更符合青春期男孩对色气美少女的憧憬。而男主奥村就是这样一个一直活在阿修福德（莉莉艾露作品男主）梦里升上高中的中二少年。如果没有女主天乃莉莉莎，他将会是个合格的自宅警备家里蹲吧（笑）。

女主天乃莉莉莎则是莉莉艾露的狂热女粉，在“明明是女孩子怎么会喜欢这种东西”的指责中长大。如此看来，二人的相遇大概是命中注定。因莉莉艾露相遇，以COS ROM为引，摩擦出色色的高中生纯爱展开。奥村的生活简直是宅友们的梦！虽然有“莉莉艾露是「俺の嫁」”的豪语，但COS莉莉艾露的学妹也是莉莉艾露没错（只要有爱就没问题）！

剧情方面，一说校园恋爱废部危机大家都能猜个八九不离十，但以ROM的拍摄为明、天乃莉莉莎作为COSER的成熟为暗的主线发展，配上喜闻乐见的傲娇幼驯染以及老师の过去（魔乳赛高！），这样的轻喜剧要素以漫画的形式糅合在一起，再加上作者对comiket的细致刻画，让读者直面COS的魅力，也直面莉莉莎的魅力。

作者的分镜也很有特色。以完整的COSER形象作为近景，仰视的视角聚焦在下半乳，光洁的小腹后是略显杂乱的漫研活动室，鲜明对比下一股热浪直冲脑门（雾）。这样的镜头，用来画色色的女孩子不是天造地设？而作者对漫画中三次元与二次元的处理也十分巧妙。三次元的天乃莉莉莎是个略显拘谨但渴望交流的短发眼镜娘，穿上C服以后却变得热情开放（不许变身！），仿佛找到了真物。至于莉莉艾露使得幼驯染对男主产生感情以及从COS界top0位置隐退的老师的故事，抑或是漫研活动室后续人物的登场，就有待大家去细细品味了。

言而总之，«2.5次元的诱惑»这本漫画不失为放松的一味好料。唯一美中不足的是，国内的热度不大，搬运工作似乎不是那么及时。如果有大大愿意持续翻译就再好不过了。
