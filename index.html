好看的皮囊男性过敏，有趣的女人水路两栖

发表于 2023-05-31 更新于 2023-12-06 分类于 漫研 本文字数： 729 阅读时长 ≈ 1 分钟

这是一部从内容到背景都非常戏谑的作品。最初的四格漫画，是为了宣传自己所作的V曲而创作。出于对“日常系女孩群像漫画”的创作愿望，作者ビーノ构思了几位符号化的主要角色。笨蛋田中、死宅菊地、机器人鹭宫、萝莉、死正经、有病、百合等等，这些角色身上或多或少都能看到作者自身的映射——人云亦云的中二、体弱多病而少年老成。而作品唯一指定男主早稻田老师，其原型低所得P，早在动画播出数年前就和作者ビーノ注册结婚，现如今二人孩子都快两岁了。联系一下作品剧情，原来死宅是作者的自设啊（笑）。

发表于 2023-05-31 更新于 2023-12-06 分类于 漫研 本文字数： 709 阅读时长 ≈ 1 分钟

作为作者野村美月的成名作，《“文学少女”》仅一卷就让我产生了码字的冲动，但真到了写短评的时候，却又不知从何说起。

许多年前我是读过《人间失格》的。只不过，那时的我更多的是把“读日本文学”当做标榜自己的工具。显然，十几岁的年纪最多允许我阅读村上村树，太宰治于我而言还是太早了些。那时的我眼中，大庭叶藏就是个很倒霉但很酷的富哥们。我不懂什么是讨好型人格，不懂为什么他对“被女人迷上”和“成为一个了不起的画家”深信不疑。阅历尚浅的我，只会在作文时文邹邹地引用一句：“我已丧失做人的资格。我已经彻底变成一个非人了。”

人鱼之森

发表于 2023-02-21 更新于 2023-12-06 分类于 漫研 本文字数： 169 阅读时长 ≈ 1 分钟

高桥留美子先生的作品大都有着厚重的基调，而动画用涌太五百年一生中的几个剪影，展现了先生观察生活、描写生活的底力。南海有鲛人，对月流珠，涌太的痛苦不止来自永生，更源于背后衍生出的“情”。君生我未生，我生君已老。君恨我生迟，我恨君生早。蹉跎了几任妻子的一生，涌太终是遇到了真鱼。虽得长相厮守，却不是挚爱，个中滋味，不细细品味作品，更与谁人说。
